# LocalChat web app

This folder contains the React and TypeScript interface for LocalChat.

```bash
npm install
npm run dev
```

The development server proxies `/api` to `http://127.0.0.1:8000`. Use `npm run lint`, `npm run test`, and `npm run build` before submitting changes. The complete installation, architecture, privacy, and deployment documentation is in the project-level `README.md`.
